### 1. Brand Constants Module

    get_color_palette,

### 2. Intro/Outro (Remotion)

The Remotion intro and caption overlays automatically use Cronduit colors:

```python
from verticals.remotion import render_intro, render_caption_overlay

# Automatically uses #111111 (dark bg) + #00E5C0 (teal accent)
intro = render_intro(
    title="Your Video Title",
    niche="AI",
    out_dir=out_dir,
    # Colors default to Cronduit brand, but can be customized:
    # accent_color="#YOUR_COLOR",
    # bg_color="#YOUR_COLOR",
)

# Captions use teal accent by default
captions = render_caption_overlay(
    words=caption_data,
    duration_seconds=30,
    out_dir=out_dir,
    # accent_color="#YOUR_COLOR",  # defaults to #00E5C0
)
```

### 3. Thumbnails

Thumbnail text overlays now use the Cronduit light gray color (#E8E8E8) with black drop shadows for maximum contrast:

```python
from verticals.thumbnail import generate_thumbnail

thumbnail = generate_thumbnail(draft, out_dir)
# Uses COLOR_GRAYSCALE_LIGHT for text, dark backgrounds generated by Gemini Imagen
```

### 4. Custom Color Usage

To override brand colors for specific projects or A/B testing:

```python
from verticals.brand import hex_to_rgb, rgb_to_hex, get_color_palette

# Convert colors
rgb = hex_to_rgb("#00E5C0")  # → (0, 229, 192)
hex_color = rgb_to_hex((0, 229, 192))  # → "#00E5C0"

# Get full palette

```

## Customization

### Override Brand Colors

To use different colors for a specific video, pass custom values:

```python
# Custom accent color
render_intro(
    title="...",
    accent_color="#FF6B6B",  # Custom color instead of teal
    bg_color="#1A1A1A",       # Custom background
    out_dir=out_dir,
)
```

### Create Alternative Palettes

Add new themes to `brand.py`:

```python
def get_color_palette(theme: str = "default") -> dict:
    if theme == "custom":
        return {
            "bg": "#2A2A2A",
            "primary": "#FF6B6B",
            "secondary": "#4ECDC4",
            "text": "#F0F0F0",
        }
```

## Troubleshooting

**Q: Colors look different in the output video**
- Ensure FFmpeg is using the correct color space (YUV420p for H.264)
- Verify that Remotion is rendering at 8K before downsampling

**Q: Intro looks washed out**
- Check that your Remotion installation is up-to-date: `npm update remotion`
- Verify accent color contrast ratio is ≥7:1 against dark background

**Q: Thumbnails have poor text visibility**
- Increase drop shadow offset in `thumbnail.py` if needed
- Consider using outline text instead of drop shadow for better legibility

**Q: Different accent colors for different videos**
- Pass `accent_color` parameter to `render_intro()` and `render_caption_overlay()`
- Document the override reason in the draft metadata
